# TestSequencer
## Roles
- superadmin
  - rights :
    - create users
    - create products
- admin
  - rights : 
    - enable integrations
    - configure integrations
    - debug integrations
    - configure products
    - connect subtest logic
    - assign products to technicians
- business owner
  - rights :
    - view products currently under test
    - create dashboards
    - view dashboards
- technician
  - rights :
    - view assigned products currently under test
    - perform subtest steps
